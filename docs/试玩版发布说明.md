# 《晚自习之后》0.9.0 策略事件版

在线地址：<https://rikka060921.github.io/After-the-Final-Bell/>

## 试玩目标

本版本用于验证三件事：

1. 玩家是否愿意把序章的关系选择带入四周日程，而不是把结局当作一次性奖励。
2. 日程、座位传页、句子拼装和一模能否共同解释结果，而不是提供直接的“正确按钮”。
3. 第二章的家庭、留言与公交流程能否把高中压力自然延伸到校外空间。

## 完整可玩边界

- 标题页、模式选择和姓名输入。
- 序章《错题本里的十分钟》及五种阶段结局。
- 第一章《座位表》的四周排程、每周六次主动安排、十二个执行事件、四次三轮压力挑战、三个核心交互和一模。
- 第二章《两份答案》的成绩单解释、十八字留言与错峰公交。
- 第二章通关总结和第三章复印件钩子。

第二章结束即为本次试玩的正式终点。第三章《纸页的背面》到第八章成年篇仍在制作，不应把当前版本理解成完整故事或“已完成 60%”。

## 已知内容边界

- 现有素材为三张校园背景和一张周棠立绘，尚无其他角色立绘与事件 CG。
- 没有 BGM、配音或环境音文件，只有程序生成的短提示音。
- 第二章是机制与叙事纵向切片，家庭场景和异步通信仍会继续扩写。
- 存档仅在当前浏览器本地保存，没有账号、云同步或遥测。
- 页面可在手机浏览器游玩，但暂未封装为原生应用或可安装 PWA。

## 发布检查

每次公开版本必须满足：

```powershell
npm ci
npm run check:release
```

- 全部领域、剧情图、存档与内容完整性测试通过。
- TypeScript 严格检查通过。
- Vite 生产构建通过。
- 标题页、章节目录、继续游戏和试玩终点在桌面浏览器可访问。
- 390×844 手机视口不出现阻断操作的溢出。
- GitHub Pages 部署完成，在线链接可以加载背景、脚本和样式。
- 下载构建解压后，Windows 用户可通过 `tools/start-demo.bat` 启动本地服务器。
