# After-the-Final-Bell
《After the Final Bell（晚自习之后）》是一部以河南县城重点高中为舞台的青春恋爱视觉小说。距离一模只剩二十一天，试卷、排名、家长期望与无休止的晚自习，几乎填满了高三生活。只有放学铃响后的十分钟，才真正属于你和周棠。你们可以交换错题、分享疲惫，在走廊尽头偷偷见面，也可能因为压力、误解和过度依赖渐行渐远。游戏拥有学业、精力、压力、自主、亲密、风险与逆反等相互影响的状态；学习越久并不会必然叛逆，失去休息和选择才会让情绪越过红线。每次决定都在改变成绩、关系与未来：是把喜欢变成逃避现实的出口，还是成为彼此继续前进的力量？从校服到婚纱并非一句承诺，而是两个人在高考、成长与漫长岁月里共同写出的答案。在这场没有标准答案的青春里，甜蜜能否战胜高考倒计时，结局最终将由你的每一次选择决定。
