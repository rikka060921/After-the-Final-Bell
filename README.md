# 《晚自习之后》

一个关于河南县中生活、学业压力与关系成长的浏览器 Galgame。当前 `0.7.0` 是核心循环重制版：玩家可以从正式序章一路玩到第二章结尾，并把序章承诺、第一章日程、一模结果和第二章选择连续带入存档。

## 在线试玩

[打开《晚自习之后》公开试玩版](https://rikka060921.github.io/After-the-Final-Bell/)

试玩数据只保存在当前浏览器的 `localStorage`，项目没有账号、服务器或数据上传功能。清理站点数据或更换浏览器会失去本地存档。

## 当前可玩内容

- 序章《错题本里的十分钟》：44 个剧情节点、7 次关键选择、21 个选项与 5 种阶段结局
- 第一章《座位表》：四周、每周十四个时间窗口与六次主动安排，以及承诺改约和周棠独立行动
- 每周三个计划执行事件：日程会改变场景文本，现场选择继续影响学业、关系、习惯和后续事实
- 6×8 座位传页、两至三段句子拼装、纸页处理和四阶段一模
- 第二章《两份答案》：成绩单解释、真实 18 字留言和错峰公交
- 第二章结束后的“第17题复印件”第三章钩子
- 故事模式与隐藏选项提示的县中模式
- 六格错题本、承诺账本、对话回看、已读对白跳过
- 自动存档、三个手动档位、快速存读档和 v1—v4 存档迁移
- 键盘操作、阅读设置、桌面与手机响应式布局
- 通关总结生成与复制分享

第三章至成年篇仍在开发，当前五种序章结局不是正式版最终结局。完整内容边界见[试玩版发布说明](docs/试玩版发布说明.md)。

## 本地开发

需要 Node.js 20.19 或更高版本；Node.js 22 至少需要 22.12。

```powershell
npm install
npm run dev
```

开发服务器默认地址由终端输出，通常为 `http://127.0.0.1:5173`。

## 发布检查与构建

```powershell
npm run check:release
```

命令会依次执行全部自动测试、类型检查和生产构建。生产文件生成在 `dist/`，可以运行：

```powershell
node tools/serve.mjs
```

然后访问 `http://127.0.0.1:8765`。推送 `main` 后，GitHub Actions 会复查并部署 GitHub Pages，同时生成可下载的构建产物。

Windows 下载版解压后可以直接双击根目录的 `START_DEMO.bat`；服务器确认启动后会自动打开浏览器。该方式仍需要本机安装 Node.js，在线试玩则不需要安装任何软件。

## 操作

- 鼠标点击 / `Enter` / `Space`：继续对话
- 数字键 `1`—`3`：选择选项
- `S`：快速保存
- `L`：读取存档
- `Esc`：打开或关闭菜单
- 第一章：点击时间格安排活动；锁定承诺需要在剧情中协商改约

```text
src/chapter-two/    第二章成绩单、留言、公交与结果回写

assets/images/      当前背景与角色图
tools/serve.mjs     生产构建静态服务器
docs/               内容大纲、技术架构、进度和发布说明
```
