# 《晚自习之后》

一个关于河南高中生活、学业压力与秘密恋爱的浏览器 Galgame。当前 `0.2.0` 版本保留了完整试玩序章，并已进入正式版工程化开发。

## 本地开发

需要 Node.js 20.19 或更高版本；Node.js 22 至少需要 22.12。

```powershell
npm install
npm run dev
```

开发服务器默认地址由终端输出，通常为 `http://127.0.0.1:5173`。

## 检查与构建

```powershell
npm run typecheck
npm test
npm run build
```

生产文件会生成在 `dist/`。构建后可以运行：

```powershell
node tools/serve.mjs
```

然后访问 `http://127.0.0.1:8765`。

## 当前可玩内容

- 约 10—15 分钟的完整序章
- 44 个剧情节点、7 次关键选择、21 个选项
- 学业、精力、压力、自主、亲密、暴露风险、逆反与隐藏共担状态
- 5 种可达结局
- 自动存档、旧试玩存档迁移、快速存读档、对话回看、键盘操作和阅读设置

## 操作

- 鼠标点击 / `Enter` / `Space`：继续对话
- 数字键 `1`—`3`：选择选项
- `S`：快速保存
- `L`：读取存档
- `Esc`：打开或关闭菜单

## 项目结构

```text
index.html          页面结构
styles.css          界面与响应式样式
src/                TypeScript 游戏运行时、剧情、状态和存档
tests/              剧情图、数值、结局和存档测试
assets/images/      现有背景与角色图
tools/serve.mjs     生产构建静态服务器
docs/               剧情设计、技术架构与开发进度
```
