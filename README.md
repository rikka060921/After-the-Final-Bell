# 《晚自习之后》

一个关于河南高中生活、学业压力与关系成长的浏览器 Galgame。当前 `0.4.0` 版本包含正式序章与第一章“座位表”的可玩纵向切片。

## 本地开发

需要 Node.js 20.19 或更高版本；Node.js 22 至少需要 22.12。

```powershell
npm install
npm run dev
```

开发服务器默认地址由终端输出，通常为 `http://127.0.0.1:5173`。

## 检查与构建

```powershell
npm run typecheck
npm test
npm run build
```

生产文件会生成在 `dist/`。构建后可以运行：

```powershell
node tools/serve.mjs
```

然后访问 `http://127.0.0.1:8765`。

## 当前可玩内容

- 约 10—15 分钟的完整序章与四周第一章切片
- 44 个剧情节点、7 次关键选择、21 个选项
- 学业、精力、压力、自主、亲密、暴露风险、逆反与隐藏共担状态
- 5 种可达结局
- 故事模式与隐藏选项提示的县中模式
- 六格可操作错题本与承诺账本
- 结局生成第一章开局档案，并把承诺映射为真实日历占用
- 每周十四格日程、承诺改约、周棠独立行动与定性周复盘
- 座位视线传页、两至三段句子拼装与四阶段一模结算
- 第17题完整页／残片／压痕三种章末呈现
- 自动存档、v1/v2/v3 存档迁移、快速存读档、对话回看、键盘操作和阅读设置

## 操作

- 鼠标点击 / `Enter` / `Space`：继续对话
- 数字键 `1`—`3`：选择选项
- `S`：快速保存
- `L`：读取存档
- `Esc`：打开或关闭菜单
- 第一章：点击时间格安排活动；锁定的承诺格需要在剧情中协商改约
