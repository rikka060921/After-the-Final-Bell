# 《晚自习之后》试玩版

一个关于河南高中生活、学业压力与秘密恋爱的浏览器 Galgame 纵向切片。

## 运行

直接双击 `index.html` 即可游玩，不需要安装依赖。

也可以使用项目自带的零依赖静态服务器：

```powershell
node tools/serve.mjs
```

然后访问 `http://localhost:8765`。

## 试玩内容

- 约 10—15 分钟
- 3 个原创校园场景
- 7 次关键选择
- 学业、精力、压力、自主、亲密、暴露风险、逆反等状态
- 5 种可达结局
- 自动存档、快速存读档、对话回看、键盘操作和阅读设置

## 操作

- 鼠标点击 / `Enter` / `Space`：继续对话
- 数字键 `1`—`3`：选择选项
- `S`：快速保存
- `L`：读取存档
- `Esc`：打开或关闭菜单

## 项目结构

```text
index.html          页面结构
styles.css          界面与响应式样式
game.js             剧情、数值、分支及存档逻辑
assets/images/      原创背景与角色图
tools/serve.mjs     可选的本地静态服务器
docs/               正式版剧情与玩法设计文档
```

正式版剧情与系统规划见 [《正式版游戏内容大纲》](docs/正式版游戏内容大纲.md)。

本试玩中的学校、人物与事件均为虚构。
